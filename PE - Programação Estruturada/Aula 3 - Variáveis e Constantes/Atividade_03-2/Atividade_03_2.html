Exercício | Validando nomes de variáveis

Atividade 03-2

Logo abaixo estarei listando alguns nomes de variáveis e expondo se são válidos ou inválidos, neste caso, explicando a razão

valorTotal -> Válido;

valortotal -> Válido;

valor_total -> Válido;

_valor_total -> Válido

$valor_total -> Inválido, pois começa com um símbolo "$";

valor_total_1 -> Válido

1_valor_total -> Inválido, pois começa com um número;

valor-total -> Inválido, pois contém com caracter inválido "-";

function -> Inválido, pois é uma palavra reservada da linguagem;

object -> Inválido, pois é uma palavra reservada da linguagem;
