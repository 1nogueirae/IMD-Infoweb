# Atividade 01 - Enunciado

> Para praticar o conceito de composição que você acabou de aprender, tente olhar o mundo ao seu redor, outros exemplos de objetos do mundo real que são compostos por outros objetos menores. Anote tais exemplos para que possamos usá-los em outros exercícios ao longo da nossa aula.

# Resposta:

- Um carro é composto por um motor, rodas, portas, assentos, etc.
- Uma casa é composta por quartos, banheiros, cozinha, sala de estar, etc.
- Um computador é composto por CPU, memória RAM, disco rígido, placa-mãe, etc.
- Um livro é composto por capa, páginas, índice, prefácio, etc.
- Um smartphone é composto por tela, bateria, câmera, processador, etc.
- Um avião é composto por asas, motores, cabine, trem de pouso, etc.
- Um ser humano é composto por cabeça, tronco, braços, pernas, órgãos internos, etc.
- Uma bicicleta é composta por quadro, rodas, guidão, pedais, etc.
- Um jardim é composto por plantas, flores, árvores, caminhos, etc.
