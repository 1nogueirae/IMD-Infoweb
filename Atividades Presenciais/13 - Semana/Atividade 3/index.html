Este site simula um formulário de cadastro de pessoas, onde você pode inserir informações como nome, idade e cidade. Fique atento às regras de validação:

    Regras para o Nome:
  • O nome não pode estar vazio.
    Regras para a Idade:
  • A idade deve ser um número positivo entre 18 e 120.
    Regras para a Cidade:
  • A cidade deve ser selecionada.
