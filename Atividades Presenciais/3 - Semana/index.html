Exercício - 02 | Sobre & Cálculo de Bhaskara

Quem foi Bhaskara?

Bhaskara foi um matemático e astrônomo indiano que viveu no século XII. Ele é conhecido por suas contribuições significativas à matemática, incluindo a resolução de equações quadráticas e o desenvolvimento de métodos para calcular raízes quadradas.

A fórmula de Bhaskara é aplicada para resolver equações do segundo grau em diversas áreas, como matemática, física, engenharia, economia e ciências da computação, sempre que é necessário encontrar as raízes de uma equação quadrática.

  • Nome Completo: Bhaskara Akaria
  • Data de Nascimento: Ano de 1114
  • Data de Falescimento: Ano de 1185
  • Local de Nascimento: Bijapur, Índia
  • Local de Falescimento: Uijain, Índia

Como aplicar a fórmula de Bhaskara

  1. Identifique os coeficientes da equação quadrática:
    1. Observe a equação no formato ax² + bx + c = 0.
    2. Determine o valor de a (coeficiente de x²).
    3. Determine o valor de b (coeficiente de x).
    4. Determine o valor de c (termo independente).
  2. Calcule o delta:
    1. Utilize a fórmula: Δ = b² - 4ac.
    2. Calcule o quadrado de b.
    3. Multiplique 4 pelo valor de a e c.
    4. Subtraia o resultado do quadrado de b.
  3. Analise o valor de delta:
    1. Se Δ > 0, existem duas raízes reais e distintas.
    2. Se Δ = 0, existe uma raiz real e dupla.
    3. Se Δ < 0, as raízes são complexas.
  4. Calcule a raiz quadrada de delta:
    1. Utilize a função de raiz quadrada: √Δ.
    2. Se Δ for negativo, considere números complexos.
  5. Substitua os valores na fórmula de Bhaskara:
    1. x₁ = (-b + √Δ) / (2a)
    2. x₂ = (-b - √Δ) / (2a)
  6. Interprete os resultados:
    1. Verifique se as soluções são reais ou complexas.
    2. Utilize as raízes conforme o contexto do problema.
