O que é Recursividade?

Recursividade é um conceito fundamental em programação onde uma função chama a si mesma para resolver um problema. Esse método é extremamente útil para dividir tarefas complexas em partes menores e mais simples de resolver. Um exemplo clássico de recursividade é o cálculo do fatorial de um número, onde a função fatorial chama a si mesma até atingir o caso base. Para que a recursividade funcione corretamente, é fundamental definir um caso base que interrompa as chamadas recursivas, evitando loops infinitos que poderiam travar o programa.

Exemplo Prático: Inversão de Números

Uma função recursiva pode ser usada para inverter um número inteiro N. Por exemplo, inverter o número 123 resulta em 321. Veja abaixo como isso pode ser implementado em JavaScript usando recursividade, garantindo que o caso base seja sempre alcançado: 

function inverterNumero(n, resultado = 0) {
    if (n === 0) {
        return resultado;
    } else {
        return inverterNumero(
            Math.floor(n / 10), 
            resultado * 10 + (n % 10)
        );
    }
}

console.log(inverterNumero(123));
console.log(inverterNumero(456));
console.log(inverterNumero(789));

Vantagens e Desvantagens

Ao utilizar recursividade, podemos simplificar significativamente a resolução de problemas complexos, tornando o código mais legível, elegante e fácil de manter. A recursividade é especialmente útil para trabalhar com estruturas de dados hierárquicas, como árvores e grafos, onde cada elemento pode conter subelementos. No entanto, é importante considerar o impacto na performance e no uso de memória, já que chamadas recursivas podem levar a um consumo maior de recursos do sistema. Em alguns casos, uma abordagem iterativa pode ser mais eficiente em termos de memória.
